This site is automatically deployed to GitHub Pages. Any changes pushed to the main branch will be live at `https://zhaozhichen.github.io` within a few minutes.

### Local Development

To run the site locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/zhaozhichen/zhaozhichen.github.io.git
   cd zhaozhichen.github.io
   ```

2. Open `index.html` in your browser or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   ```

3. Visit `http://localhost:8000` in your browser
